# Promotional landing pages (future)

This directory is reserved for `/p/[slug]` campaign landing pages (e.g., spring discount, partner pages, paid-search landers).

Suggested structure:
- `app/p/[slug]/page.tsx` — generic template.
- `content/promos/<slug>.mdx` — copy and hero image refs.

Not built in v1. Add when you start running paid acquisition.
